# 人工智能导论 - HW02 

本仓库包含人工智能导论课程第二次作业的全部内容。

## 项目结构说明
- `导读_Voyager.md`: 任务一，关于《Voyager: An Open-Ended Embodied Agent with Large Language Models》的论文导读。
- `chatbot_deepseek.py`: 任务二，基于 DeepSeek API 开发的简易对话机器人。
- `requirements.txt`: Python 运行环境依赖列表。

## 任务一：论文导读说明
- **选择论文**: Voyager (Minecraft AI Agent)
- **辅助工具**: 使用了 DeepSeek 模型生成初稿，并结合原论文进行了人工润色。
- **核心重点**: 详细分析了自动课程表、技能库和迭代提示机制。

## 任务二：Chatbot 运行指南
1. 安装依赖：`pip install -r requirements.txt`
2. 获取 API Key：前往 DeepSeek 官网获取。
3. 运行程序：`python chatbot_deepseek.py`
4. 功能说明：支持多轮对话，输入 'q' 即可安全退出。
